# Gerry the Pirate Cat — pre-launch hype page

A static teaser site for **Gerry the One-Eyed Pirate Cat**, a children's pirate‑adventure
read‑along, setting sail **1 December 2026**.

**Live preview:** https://stuart2971.github.io/GerryThePirateCat/

Built with plain HTML, CSS and JavaScript — a countdown, a live "crew" sign‑up counter,
a few playable mini‑games, sneak‑peek artwork and a meet‑the‑crew gallery. Email sign‑ups
are stored privately in Supabase through locked‑down database functions.
